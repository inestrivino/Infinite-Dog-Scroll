## About

This project delivers exactly what the title promises: an endless stream of adorable dog photos, powered by the [Dog API](https://dog.ceo/dog-api/).

---

## Installation

The website does not work within Github pages, so to use it you will have to download and then open it in a web browser.

1. Clone the repository:

   ```bash
   git clone [url]
   ```

2. Open `index.html` in your preferred browser
